## 🗂️ KELOMPOK PROGRESS KERJA

### 🟩 FASE 1: Kualifikasi & Asesmen Awal (Pertama Kali Kontak)
- [ ] **Tanyakan bidang bisnis klien** (Jasa, Dagang, UMKM, Startup, Personal Brand, dll).
- [ ] **Tentukan tipe website yang dibutuhkan:**
  - [ ] **Website Standar:** Landing Page / Profile Company statis.
  - [ ] **Sistem Kustom/Kompleks:** Toko Online / Booking System / Sistem internal.
- [ ] *(Opsional)* Arahkan klien untuk mencoba **Kalkulator Biaya** di website portfolio Anda agar ekspektasi harga selaras sejak awal.

Gunakan daftar ini untuk mencatat kebutuhan halaman dan fitur klien secara spesifik:

- [ ] **Halaman Kustom Tambahan:**

#### 2. Sistem Kelola Konten (CMS)
- [ ] **Kelola Konten Sendiri (CMS / Dashboard Admin):** Ya / Tidak (Jika Ya, tambahkan +60% dari total biaya halaman).

#### 3. Fitur Tambahan (Add-ons)
- [ ] **Payment Gateway** (+ Rp 250.000 - Jasa setup & konfigurasi ke Midtrans/Xendit/dll)
- [ ] **Google Maps** (+ Rp 75.000 - Embed peta lokasi usaha)
- [ ] **Micro-site / LinkBio** (+ Rp 100.000 - Landing page khusus bio social media)
- [ ] **Formulir Email Otomatis** (+ Rp 100.000 - Notifikasi email otomatis via Gmail)
- [ ] **Multi-Bahasa (ID/EN)** (+ Rp 100.000 - Tombol toggle bahasa Indonesia & Inggris)
- [ ] **Setup Analytics** (+ Rp 150.000 - Integrasi Google Analytics & Search Console)
- [ ] **Live Chat Widget** (+ Rp 200.000 - Widget chat gratis tanpa batas agent)
- [ ] **SEO Dasar** (+ Rp 225.000 - Konfigurasi meta tags, sitemap, & schema)

#### 4. Level Desain
- [ ] **Simple (Gratis):** Template standar, bersih, & responsif.
- [ ] **Advance (+ Rp 350.000):** Custom Layout & Animasi interaktif premium.
